# Setup 
## Clone
```
git clone https://github.com/faridmth/nutrispark.git
```
## Install dependencies
```
npm i
```
## Start the app
```
npm run dev
```
After starting the app visit : http://localhost:3000/
## Build the app
```
npm run build
```

## Technologies Used in Nutrispark

- **Next.js**

- **Tailwind CSS**

- **TypeScript**
  
  ##  component libraries.
  - **[Shadcn/ui](https://ui.shadcn.com/)**
